## ✨ 주요 기능

- **AI 기반 생활기록부 작성**: AI를 활용한 편리한 생기부 작성 
- **학생 및 생활기록부 관리**: 학생과 작성한 생활기록부를 한 번에 관리
- **공지사항**: 서비스 업데이트 및 이벤트 정보
